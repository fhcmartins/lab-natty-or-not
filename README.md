# Natural ou Fake Natty? – IA aplicada à logística e automação

Este repositório é o meu fork do lab **“Natural ou Fake Natty? Como Vencer na Era das IAs Generativas!”** da DIO, adaptado para o contexto de logística, transporte de cargas e automação de processos com IA.

## Objetivos do projeto

- Explorar o limite entre conteúdo real e conteúdo gerado por IA (texto, imagem, áudio e vídeo).
- Criar um “laboratório” de IA generativa focado em casos de uso de negócios (logística, atendimento e automação).
- Orquestrar fluxos de criação de conteúdo usando **n8n** + APIs de IA + integrações com WhatsApp/Uzap.

## Estrutura do repositório

- `prompts/` – coleções de prompts usados para gerar textos, imagens e roteiros.
- `n8n-workflows/` – exports dos fluxos usados para automatizar criação e publicação de conteúdo.
- `outputs/` – exemplos de conteúdos **Natural** (reais) e **Fake Natty** (IA), com explicação rápida de cada um.
- `docs/` – anotações sobre arquitetura, decisões técnicas e próximos passos.

## Como participar do desafio

1. Fork deste repositório (já feito).
2. Personalize este README com seus experimentos e resultados.
3. Gere conteúdos com IAs generativas e marque quais são “Natural” e quais são “Fake Natty”.
4. Publique no GitHub e compartilhe nas redes com a hashtag **#LabDIONattyOrNot**.
